```html
<!DOCTYPE html>
<html>
  <head>
    <title>A Basic HTML Document</title>
    <meta name="description" content="An example of a basic HTML document.">
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>
      HTML is a markup language used to describe the
      structure of text to a web browser.
    </p>
  </body>
</html>
```
